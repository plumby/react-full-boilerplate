# react-full-boilerplate

A boilerplate project for React, running on Node

# Added and configured

* react
* redux
* react-redux
* react-router (v4)
* react-redux-form
* redux-saga
* eslint
* redux-devtools
* css modules
* webpack
* webpack-dashboard
* express
* uuid

Includes mock API with Promises to test server calls

# Added but not yet used

* mocha
* chai
* sinon
* sinon-chai
* enzyme
* react-addons-test-utils
